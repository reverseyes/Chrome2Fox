# Chrome2Fox

Firefox extension to install Chrome Web Store extensions directly on Firefox.

## Quick Start

1. Load in Firefox: `about:debugging#/runtime/this-firefox` → "Load Temporary Add-on"
2. Select `manifest.json`
3. Visit Chrome Web Store and click "Add to Firefox"

## Requirements

- Firefox Developer Edition or Nightly
- Or set `xpinstall.signatures.required` to `false` in `about:config`
